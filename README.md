# bigdata-health-management-system
### How to Run
1. Upload data: `bash scripts/data_upload.sh`  
2. Run ingestion: `spark-submit scripts/data_ingestion.py`  
3. Feature engineering: `spark-submit scripts/feature_engineering.py`
